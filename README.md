# CRUD Query App

This project is a React Native Expo application built using TanStack Query and JSONPlaceholder.

## Features

- Fetch posts (GET)
- Create posts (POST)
- Update posts (PUT)
- Patch post titles (PATCH)
- Delete posts (DELETE)
- Filter posts by User ID

## Technologies Used

- React Native (Expo)
- TanStack Query
- JSONPlaceholder API
